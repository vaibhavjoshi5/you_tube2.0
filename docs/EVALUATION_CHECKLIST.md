# ElevanceSkills Evaluation Checklist

## Task 1: Comments

- [ ] Post English, Hindi and one South Indian language comment
- [ ] Confirm city is displayed
- [ ] Confirm a symbol-containing comment is rejected
- [ ] Translate each test comment
- [ ] Like and unlike a comment
- [ ] Dislike from two different users and confirm automatic deletion

## Task 2: Downloads

- [ ] Free user downloads one video
- [ ] Second same-day download is blocked
- [ ] Downloads page contains the history item
- [ ] Paid user downloads multiple videos

## Task 3: Plans

- [ ] Free playback stops at 5 minutes
- [ ] Bronze playback stops at 7 minutes
- [ ] Silver playback stops at 10 minutes
- [ ] Gold playback is unlimited
- [ ] Razorpay test payment succeeds
- [ ] Plan is updated only after signature verification
- [ ] Invoice email and PDF attachment arrive

## Task 4: Location and authentication

- [ ] Location permission resolves exact city and state
- [ ] Manual location fallback works
- [ ] Southern state sends email OTP
- [ ] Other state sends mobile OTP
- [ ] South India at 10:00–11:59 IST receives light theme
- [ ] All other cases receive dark theme

## Task 5: Gestures

- [ ] Double tap right seeks +10 seconds
- [ ] Double tap left seeks -10 seconds
- [ ] Single tap center toggles playback
- [ ] Triple tap center opens the next video
- [ ] Triple tap left scrolls to comments
- [ ] Triple tap right closes or opens the safe exit screen

## Task 6: Calls

- [ ] Two users join with the same room link
- [ ] Remote audio and video work
- [ ] YouTube browser tab can be selected for screen sharing
- [ ] Camera stream resumes after sharing stops
- [ ] Recording saves locally as a playable WebM file
